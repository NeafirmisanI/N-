N# is a new programming language created to teach beginners to code
</p>

## Example
```
// FizzBuzz in N#

Log = "Enter an integer"
x = Input
fizzBuzz {x}

df fizzBuzz {number} (
  a = 0
  while a < number (
    if a % 3 = 0 (
      Log = "Fizz"
    ) elif a % 5 = 0 (
      Log = "Buzz"
    )
  )
)
```
