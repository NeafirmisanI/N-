## Example
```
// FizzBuzz in N#

print "Enter an integer"
var x = Input
fizzBuzz {x}

df fizzBuzz {number} (
  var a = 0
  while a < number (
    if a % 3 = 0 (
      print "Fizz"
    ) elif a % 5 = 0 (
      print "Buzz"
    )
  )
)
```
